# 3055 - Nota Esquecida

## [Descrição](https://www.beecrowd.com.br/judge/pt/problems/view/3055)

## Solução

Dado que a média $M$ é tal que $M = \frac{A + B}{2}$ e são dados os valores de $A$ e $M$, para descobrirmos $B$ fazemos

$$
\begin{split}
M & = \frac{A + B}{2} \\
2M & = A + B \\
B & = 2M - A
\end{split}
$$
